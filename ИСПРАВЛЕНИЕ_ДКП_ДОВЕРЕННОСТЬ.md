# ✅ ИСПРАВЛЕНО: ДКП и Доверенность

## 🔍 Проблемы:

### 1. ❌ В ДКП не заполняются {{VYPUSK}} и {{PRICE_EUR}}
### 2. ❌ После доверенности меняются ФИО и паспорт клиента

## ✅ Что исправлено:

### 1. ДКП - Добавлены {{VYPUSK}} и {{PRICE_EUR}}

**Проблема:**
В mapping не было плейсхолдеров {{VYPUSK}} и {{PRICE_EUR}} (с большими буквами)

**Было в mapping:**
```python
"REGISTR": regdate,  # есть
"regdate": regdate,  # есть
"Price": price,      # есть
"price": price,      # есть
"price_eur": price_eur,  # есть (с маленькими)
# НО НЕТ: VYPUSK и PRICE_EUR
```

**Стало в mapping:**
```python
"REGISTR": regdate,       # первая регистрация мм.гггг
"VYPUSK": regdate,        # ← ДОБАВЛЕНО! Для старых шаблонов
"regdate": regdate,       # для совместимости
"Price": price,           # PRICE_EUR
"price": price,           
"price_eur": price_eur,   # с маленькими
"PRICE_EUR": price_eur,   # ← ДОБАВЛЕНО! С большими буквами
"prtotal": prtotal,       # PRICE_EUR + 50
```

**Результат:**
- ✅ {{VYPUSK}} заполняется (датой первой регистрации)
- ✅ {{PRICE_EUR}} заполняется (ценой в EUR)
- ✅ Совместимость со старыми и новыми шаблонами

---

### 2. Доверенность - НЕ меняет профиль клиента

**Проблема:**
После создания доверенности ФИО и паспорт КЛИЕНТА менялись на ФИО и паспорт ПОВЕРЕННОГО (получателя платежа)!

**Было:**
```python
upd_fields = {
    "fio": fio,           # ← Это ФИО поверенного!
    "passport": passport, # ← Это паспорт поверенного!
    "address": address,   # ← Это адрес поверенного!
    "car_model": car_model,
    "vin": vin,
}
db_upsert_client(cid, upd_fields)
```

**Результат:** ФИО и паспорт клиента ПЕРЕЗАПИСЫВАЛИСЬ данными поверенного! ❌

**Стало:**
```python
# ВАЖНО: Доверенность на ПОЛУЧАТЕЛЯ ПЛАТЕЖА, а не на клиента
# НЕ сохраняем ФИО и паспорт поверенного в профиль клиента!
upd_fields = {
    # Только данные авто
    "car_model": car_model,
    "vin": vin,
    # История поверенных
    "last_proxy_fio": fio,  # Запоминаем кто был поверенным
    "last_company_id": company_id,
}
db_upsert_client(cid, upd_fields)
```

**Результат:**
- ✅ ФИО клиента НЕ меняется
- ✅ Паспорт клиента НЕ меняется
- ✅ Адрес клиента НЕ меняется
- ✅ Сохраняются только данные авто
- ✅ last_proxy_fio - запоминаем последнего поверенного

## 🧪 Как проверить:

### Тест 1: ДКП с {{VYPUSK}}

1. Создай ТПО с PRICE_EUR = 5000
2. Создай ДКП
3. В шаблоне используй:
   ```
   {{VYPUSK}} - дата регистрации
   {{PRICE_EUR}} - цена в EUR
   ```
4. Создай документ

**Проверь в документе:**
- {{VYPUSK}} = "01.2020" ✅
- {{PRICE_EUR}} = "5000" ✅

### Тест 2: Доверенность НЕ меняет клиента

1. Создай клиента "Иванов И.И.", паспорт "1234 567890"
2. Создай счет с получателем "Петров П.П."
3. Создай доверенность на Петрова П.П., паспорт "9876 543210"
4. Открой профиль клиента

**Проверь:**
- ФИО клиента: "Иванов И.И." ✅ (НЕ изменилось!)
- Паспорт клиента: "1234 567890" ✅ (НЕ изменился!)
- last_proxy_fio: "Петров П.П." ✅ (запомнили поверенного)
