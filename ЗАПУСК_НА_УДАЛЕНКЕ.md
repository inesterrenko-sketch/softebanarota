# 🖥️ ЗАПУСК НА УДАЛЕННОМ WINDOWS СЕРВЕРЕ

## 🎯 Цель: Запустить приложение на удаленке и зайти с твоего ПК

---

## 📋 Быстрый старт (5 минут)

### Шаг 1: Подключись к удаленному серверу

**Windows Remote Desktop (RDP):**
1. Нажми `Windows + R`
2. Введи `mstsc` и нажми Enter
3. Введи IP адрес сервера
4. Введи логин/пароль
5. Подключись

### Шаг 2: Перенеси файлы на сервер

**Самый простой способ - копипаст:**
1. На своем ПК открой архив `ContractFill_FINAL_v3.zip`
2. Выдели все файлы → Ctrl+C (скопировать)
3. В окне RDP (на сервере) создай папку `C:\ContractFill`
4. Открой эту папку и нажми Ctrl+V (вставить)
5. Файлы скопируются на сервер!

**Альтернатива - через общий диск:**
1. Перед подключением RDP нажми "Показать параметры"
2. Вкладка "Локальные ресурсы" → "Подробнее"
3. Поставь галочку на диск C:
4. Подключись
5. На сервере открой "Этот компьютер" → увидишь свой диск
6. Скопируй архив с твоего диска на сервер

### Шаг 3: Установи Python на сервере (если нет)

1. **Проверь, есть ли Python:**
   - На сервере открой командную строку: `Windows + R` → `cmd`
   - Введи: `python --version`
   - Если видишь версию Python → переходи к Шагу 4
   - Если ошибка → продолжай установку

2. **Скачай Python:**
   - Открой браузер на сервере
   - Перейди: https://www.python.org/downloads/
   - Скачай Python 3.11 или новее

3. **Установи Python:**
   - Запусти установщик
   - **ВАЖНО:** Поставь галочку "Add Python to PATH"
   - Нажми "Install Now"
   - Дождись окончания

### Шаг 4: Запусти приложение в сетевом режиме

1. **Открой папку с приложением:**
   - На сервере открой `C:\ContractFill`

2. **Запусти батник для сети:**
   - Двойной клик на `START_NETWORK.bat`
   - Откроется окно с надписью:

```
╔═══════════════════════════════════════════════════════════╗
║  ПРИЛОЖЕНИЕ ЗАПУСКАЕТСЯ В РЕЖИМЕ СЕТИ                    ║
║                                                           ║
║  Доступ с этого компьютера:                              ║
║  → http://localhost:8000                                  ║
║                                                           ║
║  Доступ с других компьютеров в сети:                     ║
║  → http://192.168.1.XXX:8000                             ║
║                                                           ║
║  Для остановки: Ctrl+C                                   ║
╚═══════════════════════════════════════════════════════════╝

🚀 Запуск сервера...
```

3. **Запомни IP адрес!**
   - Например: `http://192.168.1.100:8000`
   - Это адрес твоего сервера

### Шаг 5: Открой порт в файерволе Windows

**Если не работает доступ извне:**

1. На сервере нажми `Windows + R` → введи `wf.msc`
2. Слева выбери "Правила для входящих подключений"
3. Справа нажми "Создать правило"
4. Выбери "Для порта" → Далее
5. Выбери "TCP" → "Определенные локальные порты" → введи `8000`
6. Далее → "Разрешить подключение" → Далее
7. Оставь все галочки → Далее
8. Имя: "ContractFill" → Готово

### Шаг 6: Открой приложение со своего ПК

1. **На своем ПК (не на сервере!) открой браузер**
2. **Введи адрес:**
   ```
   http://IP_СЕРВЕРА:8000
   ```
   Например: `http://192.168.1.100:8000`

3. **Готово!** Должно открыться приложение! 🎉

## 🔄 Автозапуск при загрузке сервера

Чтобы приложение запускалось автоматически при включении сервера:

### Способ 1: Планировщик заданий (рекомендуется)

1. На сервере нажми `Windows + R` → введи `taskschd.msc`
2. Справа "Создать простую задачу"
3. Имя: "ContractFill автозапуск"
4. Триггер: "При запуске компьютера"
5. Действие: "Запустить программу"
6. Программа: `C:\ContractFill\START_NETWORK.bat`
7. Готово

### Способ 2: Автозагрузка

1. Нажми `Windows + R` → введи `shell:startup`
2. Создай ярлык на `START_NETWORK.bat`
3. Перетащи ярлык в открывшуюся папку

## 📡 Узнать IP адрес сервера

Если забыл IP адрес сервера:

**На сервере:**
1. Открой командную строку: `Windows + R` → `cmd`
2. Введи: `ipconfig`
3. Найди "IPv4-адрес" → это твой IP

**Пример вывода:**
```
Адаптер Ethernet:
   IPv4-адрес. . . . . . . . . : 192.168.1.100
```

Твой IP: **192.168.1.100**
Адрес для браузера: **http://192.168.1.100:8000**

## 🌐 Доступ из интернета (опционально)

Если нужен доступ не только из локальной сети, но и из интернета:

### Вариант 1: Проброс портов на роутере

1. Зайди в настройки роутера (обычно http://192.168.1.1)
2. Найди раздел "Проброс портов" / "Port Forwarding"
3. Добавь правило:
   ```
   Внешний порт: 8000
   Внутренний IP: IP_сервера (например 192.168.1.100)
   Внутренний порт: 8000
   Протокол: TCP
   ```
4. Сохрани

5. Узнай внешний IP:
   - На сервере открой https://myip.com
   - Запомни IP (например: 85.123.45.67)

6. Доступ из интернета:
   ```
   http://ВНЕШНИЙ_IP:8000
   ```
   Например: `http://85.123.45.67:8000`

### Вариант 2: Ngrok (бесплатный туннель)

1. Скачай ngrok: https://ngrok.com/download
2. На сервере запусти:
   ```
   ngrok http 8000
   ```
3. Получишь адрес вида: `https://abc123.ngrok.io`
4. Этот адрес работает из любой точки мира!

## ⚠️ Решение проблем

### Проблема 1: "Это приложение заблокировано"

**Решение:**
1. Правой кнопкой на `START_NETWORK.bat`
2. Свойства
3. Внизу галочка "Разблокировать"
4. Применить → OK

### Проблема 2: Не открывается с другого ПК

**Проверь:**
1. Приложение запущено на сервере? (окно должно быть открыто)
2. Файервол открыт? (см. Шаг 5)
3. Оба компьютера в одной сети?
4. Правильный IP адрес? (не 127.0.0.1, а реальный IP)

**Быстрая проверка:**
- На сервере открой браузер → `http://localhost:8000` (должно работать)
- Если работает на сервере, но не с твоего ПК → проблема в файерволе

### Проблема 3: Python не найден

**Решение:**
1. Переустанови Python
2. При установке обязательно поставь галочку "Add Python to PATH"
3. Перезагрузи сервер

### Проблема 4: Порт 8000 занят

**Решение:**
1. Открой `START_NETWORK.bat` в блокноте
2. Найди строку с `--port 8000`
3. Замени на `--port 8080` (или любой другой)
4. Сохрани
5. Теперь адрес будет: `http://IP:8080`
