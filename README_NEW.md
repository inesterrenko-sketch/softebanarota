# 🚀 ContractFill - Premium Edition

## ✨ Новые возможности

### 🎨 Современный дизайн
- **Стеклянный эффект** (glass-morphism) для навбара
- **3D карточки** с анимациями при наведении
- **Градиентные кнопки** с плавными переходами
- **Анимированный фон** с частицами
- **Preloader** при загрузке страницы
- **Пользовательская прокрутка** с градиентом
- **Hover эффекты** и свечение
- **Адаптивный дизайн** для всех устройств

### 📋 Полное сохранение данных из договоров
Все поля автоматически распознаются и сохраняются:
- ✅ CLIENT_FIO, ADDRESS, PASSPORT
- ✅ PHONE, ORGAN, VYDAN
- ✅ CAR, CAR_ALL, CAR_RUB
- ✅ year car, Color, Probeg
- ✅ Engine, OBEM, Delivery
- ✅ vin, CONTRACT_NO, CONTRACT_DATE
- ✅ CUST_RUB (таможенная пошлина)

### 🔄 Автоматическое подтягивание данных
- Данные из договора автоматически подставляются во все документы
- ДКП, утильсбор, квитанция, ТПО - все заполняется автоматически
- Приоритет: форма → последний договор → база данных

### 🌐 Развертывание в сети
- **Nginx** конфигурация для reverse proxy
- **Systemd service** для автозапуска
- **Docker + Docker Compose** для простого развертывания
- Поддержка SSL/HTTPS
- Готово для локальной сети и интернета

## 🚀 Быстрый старт

### Локально (Windows)
```bash
START.bat
```

### Локальная сеть
```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```
Доступ: `http://ваш-ip:8000`

### Docker (рекомендуется)
```bash
docker-compose up -d
```
Доступ: `http://localhost`

## 🎯 Основные функции

### Управление клиентами
- Создание и редактирование клиентов
- Полный профиль с историей документов
- Автоматическое сохранение всех данных
- Поиск и фильтрация

### Генерация документов
- **Договоры** - с автоматическим распознаванием всех полей
- **ДКП** - немецкий договор купли-продажи
- **Утильсбор** - письмо Минской таможни
- **Квитанция** - платежная квитанция
- **ТПО** - таможенный приходный ордер
- **Доверенность** - на представление интересов
- **Счет** - счет на оплату

### Управление компаниями
- Добавление своих компаний
- Редактирование данных компаний
- Автоматическая синхронизация в документах

## 🛠 Технологии

- **Backend**: FastAPI + Python 3.11
- **Frontend**: Bootstrap 5 + Custom CSS3
- **Database**: SQLite
- **Documents**: python-docx
- **Server**: Uvicorn + Nginx
- **Deploy**: Docker + Systemd
