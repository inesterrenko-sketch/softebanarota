## 📞 ПОМОЩЬ

Если что-то не работает:

1. **Локально не запускается?**
   → Проверьте `START.bat` / `START_SIMPLE.bat`

2. **Не видно из сети?**
   → Проверьте firewall / router

3. **Docker не работает?**
   → `docker-compose logs -f`

4. **Nginx ошибки?**
   → `sudo nginx -t`
