## 🔧 Что случилось:

Вы получили ошибку при создании счета:
```
Package not found at 'invoice_...docx'
```

**Причина:** Шаблон `invoice_template.docx` был пустым (0 байт)

## ✅ Исправлено:

Создан минимальный рабочий шаблон счета с полями:
- CONTRACT_NO - номер договора
- CONTRACT_DATE - дата договора
- CLIENT_FIO - ФИО клиента
- CAR_MODEL - марка автомобиля
- PAYMENT_PURPOSE - назначение платежа
- AMOUNT - сумма
- RECIPIENT_FIO - получатель
- BANK_NAME - банк
- BANK_BIK - БИК
- BANK_RS - расчетный счет
- BANK_KS - корреспондентский счет

## 📝 Как использовать свой шаблон:

### Вариант 1: Заменить файл
1. Создайте свой шаблон счета в Word
2. Вставьте плейсхолдеры: `{{CONTRACT_NO}}`, `{{CLIENT_FIO}}` и т.д.
3. Сохраните как `invoice_template.docx`
4. Замените файл в папке `app/data/`

### Вариант 2: Использовать текущий и доработать
1. Откройте `app/data/invoice_template.docx`
2. Отредактируйте оформление
3. Добавьте логотип, реквизиты и т.д.
4. Сохраните

## 🎯 Как добавить свои шаблоны:

### Через интерфейс:
1. Откройте приложение
2. Перейдите в "Админ" → "Загрузить шаблон"
3. Выберите файл .docx
4. Укажите название

### Вручную:
1. Положите файл в папку `app/data/`
2. Переименуйте в нужное имя (см. таблицу выше)
3. Перезапустите приложение

## ⚠️ Важные плейсхолдеры:

### Счет (invoice):
- `{{CONTRACT_NO}}`, `{{CONTRACT_DATE}}`
- `{{CLIENT_FIO}}`, `{{CAR_MODEL}}`
- `{{PAYMENT_PURPOSE}}`, `{{AMOUNT}}`
- `{{RECIPIENT_FIO}}`, `{{BANK_NAME}}`
- `{{BANK_BIK}}`, `{{BANK_RS}}`, `{{BANK_KS}}`
- `{{BANK_INN}}`, `{{BANK_KPP}}`

### ТПО:
- `{{FIO}}`, `{{ADDRESS}}`, `{{PASSPORT}}`
- `{{CAR}}`, `{{VIN}}`, `{{OBEM}}`
- `{{PRICE_RUB}}`, `{{PRICE_BYN}}`, `{{PRICE_EUR}}`

### Утильсбор:
- `{{fio}}` - родительный падеж
- `{{1fio}}` - творительный падеж  
- `{{fio1}}` - именительный падеж
- `{{company}}`, `{{inn}}`, `{{address}}`
- `{{car}}`, `{{year_car}}`, `{{vin}}`

### ДКП:
- `{{fio}}`, `{{passport}}`, `{{address}}`
- `{{company}}` - транслит
- `{{car}}`, `{{vin}}`, `{{obem}}`
- `{{REGISTR}}` - первая регистрация мм.гггг
- `{{Color}}` - цвет на немецком
- `{{Price}}`, `{{prtotal}}`

### Доверенность:
- `{{FIO}}`, `{{passport}}`
- `{{DROP_ORGAN}}` - кем выдан паспорт
- `{{DROP_VYDAN}}` - дата выдачи
- `{{CAR}}`, `{{VIN}}`
- `{{COMPANY}}`, `{{Date}}`, `{{SROK}}`

## 🧪 Проверка шаблона:

После замены шаблона:
1. Откройте файл в Word
2. Проверьте, что плейсхолдеры написаны правильно: `{{FIELD}}`
3. Убедитесь, что нет опечаток
4. Создайте тестовый документ в приложении
