## ИСПРАВЛЕННЫЕ ПРОБЛЕМЫ:

### 1. ✅ Распознавание {{year_car}} в договорах

**Проблема:** В договорах используется placeholder {{year_car}}, но он не распознавался и не сохранялся в базу данных.

**Исправление:**
- В файле `app/main.py`, функция `generate` (строка ~1249):
  - Добавлено распознавание всех вариантов года: `year_car`, `year car`, `YEAR_CAR`, `YEAR CAR`, `VYPUSK`, `ГОД`, `YEAR`
  - Теперь любой из этих placeholders в договоре будет сохранен в поле `vypusk` базы данных

**Код:**
```python
"vypusk": _pick_first(mapping, ["year_car", "year car", "YEAR_CAR", "YEAR CAR", "VYPUSK", "ГОД", "YEAR"]),
```

### 2. ✅ Распознавание color и probeg в договорах

**Проблема:** Поля color (цвет) и probeg (пробег) в договорах могли быть в разных регистрах и не всегда распознавались.

**Исправление:**
- В файле `app/main.py`, функция `generate` (строки ~1251-1252):
  - Добавлено распознавание всех вариантов: строчные буквы добавлены в начало списка для приоритета
  - `color`: `color`, `Color`, `COLOR`, `ЦВЕТ`
  - `probeg`: `probeg`, `Probeg`, `PROBEG`, `ПРОБЕГ`

**Код:**
```python
"color": _pick_first(mapping, ["color", "Color", "COLOR", "ЦВЕТ"]),
"probeg": _pick_first(mapping, ["probeg", "Probeg", "PROBEG", "ПРОБЕГ"]),
```

### 3. ✅ Подтягивание данных из договора во ВСЕ документы

**Проблема:** Данные из договора (year_car, color, probeg и другие) не подтягивались в другие документы (утильсбор, квитанция и т.д.).

**Исправление:**

#### 3.1. Утильсбор (util_create)
- Добавлено чтение данных из `last_contract_json`
- Приоритет подтягивания: **форма → договор → база данных**
- Данные автоматически подтягиваются:
  - ФИО, паспорт, адрес
  - Марка авто, VIN
  - **Год выпуска** (year_car) - подтягивается из договора
  - **Цвет** (color) - подтягивается из договора
  - **Пробег** (probeg) - подтягивается из договора
- Все данные сохраняются в базу данных для последующего использования

**Пример кода:**
```python
# Подтягиваем данные с приоритетом: форма → договор → база данных
year_car = (form.get("year_car") or "").strip() or contract_data.get("year_car", "") or contract_data.get("VYPUSK", "") or client.get("vypusk", "")
color = contract_data.get("color", "") or contract_data.get("Color", "") or client.get("color", "")
probeg = contract_data.get("probeg", "") or contract_data.get("Probeg", "") or client.get("probeg", "")
```

#### 3.2. Квитанция (kvit_create)
- Аналогично добавлено подтягивание данных из договора
- ФИО, паспорт, адрес теперь берутся из последнего договора

#### 3.3. Форма заполнения договора (fill.html)
- Обновлена функция `loadClient()` для подтягивания всех данных:
  - VIN, OBEM, VYPUSK (год)
  - year_car (в разных вариантах)
  - Color, color, COLOR (цвет)
  - Probeg, probeg, PROBEG (пробег)
