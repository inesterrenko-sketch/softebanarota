# ИСПРАВЛЕНИЯ УТИЛЬСБОРА И КОМПАНИЙ - 20.01.2026

## ✅ Что исправлено:

### 1. Привязка компании к клиенту

**Проблема:** Данные компании (название, ИНН, адрес) не привязывались к клиенту и не подтягивались автоматически при выборе клиента.

**Решение:**
- Добавлено поле `company_id` в базу данных клиентов
- Теперь компания привязывается к клиенту и сохраняется
- При выборе клиента данные компании автоматически подтягиваются

### 2. Выбор компании из списка в утильсборе

**Было:** Нужно было вводить данные компании вручную каждый раз

**Стало:** 
- Dropdown список компаний из `companies.json`
- Кнопка "Очистить" для сброса
- Возможность ввода вручную

**Как работает:**
1. Выбираете компанию из списка → автоматически заполняются:
   - Название компании
   - ИНН компании
   - Юридический адрес компании
2. Или вводите данные вручную

### 3. Автоматическое подтягивание данных компании

**Как работает:**

#### При выборе клиента:
```
Клиент выбран
    ↓
Загрузка данных клиента через API
    ↓
Автоматически заполняется:
  - ФИО
  - Паспорт
  - Адрес
  - Марка авто
  - Год выпуска
  - VIN
  - Название компании (из профиля)
  - ИНН компании (из профиля)
  - Адрес компании (из профиля)
  - Компания в dropdown (если привязана)
```

#### При выборе компании из списка:
```
Компания выбрана
    ↓
Загрузка данных компании
    ↓
Автоматически заполняется:
  - Название компании
  - ИНН
  - Юридический адрес
  - Сохраняется company_id (скрытое поле)
```

### 4. Сохранение company_id в профиль клиента

**Как работает:**
- При создании утильсбора с выбранной компанией, `company_id` сохраняется в профиль клиента
- При следующем выборе клиента компания автоматически выбирается в dropdown

## 📋 Структура базы данных

Добавлено поле в таблицу `clients`:

```sql
company_id TEXT NOT NULL DEFAULT ''
```

Это поле хранит ID компании из companies.json (например: "autoberg", "avtolux")

## 🎯 Workflow

### Сценарий 1: Новый клиент с компанией

1. **Создание утильсбора:**
   - Заполняете ФИО, паспорт
   - Выбираете компанию из списка → автоматически заполняются название, ИНН, адрес
   - Заполняете остальные данные
   - Генерируете документ

2. **Сохранение:**
   - ФИО, паспорт, адрес → в профиль клиента
   - company_id → в профиль клиента
   - company_name, company_inn, company_address → в профиль клиента

3. **Следующий раз:**
   - Выбираете клиента
   - Все данные + компания подтягиваются автоматически!

### Сценарий 2: Существующий клиент

1. **Выбор клиента:**
   - Выбираете клиента из списка
   - Автоматически заполняется:
     - ФИО, паспорт, адрес
     - Данные авто (марка, год, VIN)
     - Данные компании (название, ИНН, адрес)
     - Компания в dropdown (если была привязана)

2. **Редактирование:**
   - Можете изменить компанию, выбрав другую из списка
   - Можете отредактировать данные вручную

3. **Сохранение:**
   - Обновленные данные сохраняются в профиль клиента
