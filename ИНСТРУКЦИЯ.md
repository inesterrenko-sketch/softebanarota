# 🚀 БЫСТРЫЙ СТАРТ: КАК ИСПОЛЬЗОВАТЬ НОВЫЕ ВОЗМОЖНОСТИ

## 📋 Что нового?

### 1. Год, цвет и пробег автоматически сохраняются

**Как это работает:**
1. Создаешь договор, заполняешь поля:
   - `{{year_car}}` или `{{VYPUSK}}` → год выпуска
   - `{{color}}` → цвет
   - `{{probeg}}` → пробег

2. Генерируешь договор → **все данные автоматически сохраняются** в профиле клиента

3. Теперь при создании других документов (утильсбор, квитанция) эти данные **автоматически подставляются**!

### 2. Просмотр профиля клиента

**Новая функция в форме договора:**

При выборе клиента появляется синее окно:
```
┌────────────────────────────────────────────────┐
│ ℹ️ Клиент выбран! После заполнения договора   │
│   все данные сохранятся в профиле клиента.    │
│                        [Открыть профиль →]    │
└────────────────────────────────────────────────┘
```

**Нажми "Открыть профиль →"** чтобы:
- Посмотреть всю историю документов клиента
- Увидеть сохраненные данные
- Быстро перейти к созданию новых документов

### 3. Компании синхронизированы

Теперь список компаний для ДКП берется из `companies.json`.

**Как добавить новую компанию:**
1. Перейди в раздел **"Управление компаниями"** (в меню слева)
2. Добавь компанию с названием, адресом и ИНН
3. Компания **автоматически появится** в списке при создании ДКП и других документов

## 🎯 Типичные сценарии использования

### Сценарий 1: Новый клиент с нуля

1. **Создать договор:**
   - Перейди в "Создать документ"
   - Выбери шаблон договора
   - Заполни все поля, включая:
     - {{year_car}} = 2020
     - {{color}} = schwarz
     - {{probeg}} = 45000
   - Нажми "Сгенерировать .docx"

2. **Открыть профиль:**
   - После создания договора выбери клиента в выпадающем списке
   - Нажми "Открыть профиль →"
   - Проверь, что все данные сохранились

3. **Создать утильсбор:**
   - Перейди в раздел "Письмо об утильсборе"
   - Выбери клиента из списка
   - **Год выпуска, цвет и пробег подставятся автоматически!**

### Сценарий 2: Существующий клиент

1. **Выбрать клиента:**
   - Перейди в "Создать документ"
   - Выбери клиента из списка
   - Нажми "Подставить данные клиента"

2. **Проверить данные:**
   - Все данные из последнего договора автоматически заполнятся
   - Год, цвет, пробег уже будут на месте

3. **Дополнить и сохранить:**
   - Отредактируй нужные поля
   - Сгенерируй документ
   - Обновленные данные сохранятся в профиле

### Сценарий 3: Массовое создание документов

**Быстрый workflow:**

1. Договор → год/цвет/пробег сохраняются ✅
2. Утильсбор → данные подтягиваются автоматически ✅
3. Квитанция → данные подтягиваются автоматически ✅
4. ДКП → данные подтягиваются автоматически ✅

**Экономия времени:** вместо того, чтобы вводить одни и те же данные 4 раза, вводишь их один раз в договоре!

## 🔧 Технические детали

### Какие placeholders распознаются:

**Год выпуска:**
- `{{year_car}}`
- `{{year car}}`
- `{{YEAR_CAR}}`
- `{{VYPUSK}}`
- `{{ГОД}}`
- `{{YEAR}}`

**Цвет:**
- `{{color}}`
- `{{Color}}`
- `{{COLOR}}`
- `{{ЦВЕТ}}`

**Пробег:**
- `{{probeg}}`
- `{{Probeg}}`
- `{{PROBEG}}`
- `{{ПРОБЕГ}}`

### Где хранятся данные:

**База данных (clients.db):**
- `vypusk` - год выпуска
- `color` - цвет
- `probeg` - пробег
- `company_name` - название компании
- `company_inn` - ИНН
- `company_address` - юридический адрес
- `last_contract_json` - JSON с полными данными последнего договора

## ❓ FAQ

**Q: Я создал договор, но год не подтягивается в утильсбор. Почему?**
A: Убедись, что в договоре используется один из правильных placeholders (например, `{{year_car}}`). Проверь профиль клиента - там должны быть сохраненные данные.

**Q: Как посмотреть, какие данные сохранены у клиента?**
A: Перейди в раздел "Клиенты" → Найди нужного клиента → Нажми на карточку клиента. В профиле увидишь все сохраненные данные.

**Q: Можно ли изменить сохраненные данные?**
A: Да! Просто создай новый договор с обновленными данными - они автоматически перезапишут старые.

**Q: Год выпуска в договоре - это {{year_car}} или {{VYPUSK}}?**
A: Оба варианта работают! Система распознает и сохраняет оба. Используй тот, который есть в твоем шаблоне.

**Q: Как добавить новую компанию для ДКП?**
A: Перейди в меню → "Управление компаниями" → Добавь компанию. Она автоматически появится в списке при создании ДКП.
