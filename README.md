# ContractFill (web-приложение для заполнения договоров .docx)

Функции:
- **Создать документ**: выбор компании/шаблона → автогенерация формы по плейсхолдерам вида `{{KEY}}` → скачивание `.docx`
- **Запоминание клиентов**: можно выбрать клиента и быстро подставить ФИО/паспорт/адрес и т.д.
- **Счёт на оплату**: генерация счёта по шаблону `app/data/invoice_template.docx`, реквизиты можно вставлять одним текстом (автораспознавание)
- **Таможенный ордер (ТПО)**: генерация ТПО с автоматическим расчетом сумм и QR-кодом
- **Квитанция**: создание квитанций с жирным выделением важных данных
- **Письмо об утильсборе**: генерация официального письма Минской таможни с автоматической конвертацией суммы в текст прописью
- Добавление/удаление шаблонов договоров через интерфейс (загрузка `.docx`)

## Быстрый старт

```bash
python -m venv .venv
# Windows: .venv\\Scripts\\activate
# Linux/macOS: source .venv/bin/activate
pip install -r requirements.txt

uvicorn app.main:app --reload --port 8000
```

Открой: `http://127.0.0.1:8000`

## Где лежат шаблоны
- `app/data/uploads/` — загруженные/сэмпл `.docx`
- `app/data/templates.json` — база шаблонов (id, компания, поля и т.д.)
- `app/data/generated/` — сгенерированные документы
- `app/data/invoice_template.docx` — шаблон счёта
- `app/data/tpo_template.docx` — шаблон ТПО
- `app/data/kvit_template.docx` — шаблон квитанции
- `app/data/util_template.docx` — шаблон письма об утильсборе

## Данные
- `app/data/clients.db` — база клиентов (и последние суммы из договоров, если они присутствуют в заполнении)

## Формат плейсхолдеров
- В документе используй `{{CLIENT_FIO}}`, `{{CONTRACT_DATE}}` и т.п.
- Можно с пробелами в ключе: `{{year car}}`

### Плейсхолдеры для квитанции:
- `{{CLIENT NAME}}` - ФИО плательщика
- `{{ADRESS}}` - Адрес
- `{{PASSPORT}}` - Паспорт
- `{{VYDAN}}` - Кем выдан
- `{{SUMMA}}` - Сумма

### Плейсхолдеры для письма об утильсборе:
- `{{fio}}` - ФИО владельца
- `{{passport}}` - Паспорт
- `{{address}}` - Адрес
- `{{company}}` - Компания
- `{{inn}}` - ИНН
- `{{car}}` - Марка авто
- `{{year car}}` - Год выпуска
- `{{vin}}` - VIN
- `{{nomer}}` - Номер договора
- `{{bank}}` - Банк
- `{{account}}` - Расчетный счет
- `{{summa}}` - Сумма цифрами
- `{{summa text}}` - Сумма прописью (генерируется автоматически)
- `{{date number}}` - День (текущая дата)
- `{{date mouth}}` - Месяц (текущая дата)
- `{{year}}` - Год (текущая дата)

> Примечание: лучше, чтобы весь плейсхолдер был одним стилем/шрифтом (не разбивался на куски), но приложение старается заменять корректно даже если Word разбил на несколько runs.
